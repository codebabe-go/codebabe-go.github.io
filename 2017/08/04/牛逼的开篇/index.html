牛逼的开篇

发表于 2017-08-04

每个博客都应该有一个牛逼的开篇

读书时觉得域名很贵, 技术很难, 一直开不了篇
工作后发现域名没想象中那么贵, 技术没想象中那么难, 还是一直开不了篇
懒癌早期的我带薪休假了一把, 在工作期间 24 + hexo + GitHub pages算是搭建了一个自己的小窝吧

撸此代码三十年, 直到代码崩塌

宅男, 平时没啥爱好. 想着能听着万青写着牛掰的东东也不失为一种享受, 大概就是写到没代码写吧.

关于搭建本站

使用Hexo搭建个人博客 + 为部署在Github上的Hexo博客绑定个性域名 足矣

千里之行始于足下吧, 初衷就是强迫自己做一些技术沉淀, 希望不要三天打鱼两天晒网吧.

没有牛逼的技术, 可能就是爱瞎比比
